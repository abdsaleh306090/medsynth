## 📄 What is medsynth?

medsynth creates synthetic medical records. These records look real but do not belong to real people. It uses different medical formats from many regions. This helps users test healthcare software without using private information.

medsynth supports multiple languages such as Arabic and Hebrew. It can handle various layouts and schemas seen in real medical records. The software helps researchers and developers build data tools safely.

## 🛠️ Using medsynth

The main purpose of medsynth is to create medical records that look real. Here’s how to use it effectively.

### Select Locale and Language

medsynth lets you choose from many locales, including regions that use Arabic and Hebrew. Choose the one you need from the drop-down menu inside the app. This ensures records match the style of that area.

### Choose Data Volume

Set how many medical records you want to create. You can create just a few for testing or thousands for larger projects.

### Customize Variance

You can adjust how much the records vary. Higher variance means more realistic differences between records, closer to what you see in real healthcare data.

### Export Your Data

Once you generate the data, export it in common formats like CSV or JSON. These work well with most data analysis tools.
